# week1--Verilog-codes-synthesis
Synthesizable Verilog implementations of digital hardware designs and FSM-based controllers.
I have modified the previously uploaded codes to make them synthesizable on Xilinx Vivado .
One of them is a result box   [synthesis result.jpeg]  stating that synthesis was successful.
I did this by removing the elements which did not stand consistent with the hardware implementation of the code such as delays and modified them using counter run by the clock.
